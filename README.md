# Proyecto: Pipeline de Datos Masivos

## Objetivo

Transformar datos crudos de e-commerce


en información confiable para negocio.

## Capas
Raw: datos originales.

Silver: datos limpios y tipados.

Gold: datos listos para análisis.

## Preguntas de Negocio
1. ¿Cuál es el revenue mensual?
2. ¿Qué categorías venden más?
3. ¿Qué estados tienen mayores demoras?
